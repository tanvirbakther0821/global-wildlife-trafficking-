# Wildlife Trafficking Dashboard 

This project visualizes global wildlife trafficking incidents using Tableau.

## Features
- Choropleth map of incidents by country
- 4-step color gradation (lighter = fewer incidents, darker = more)
- Gray for missing data
- Interactive filters by year
- Tooltip shows:
  - Country
  - Year
  - Number of incidents
  - Average fine (USD)
  - Average imprisonment (years)

## Data
- `wildlife_trafficking.csv`: Incident data  
- `world_countries.json`: Country boundaries  

## How to Use
- Open `wildlife-trafficking-dashboard.twbx` in Tableau
- Filter by year to explore patterns
